• 주제 : 소셜미디어 + 마켓(e-commerce)
• 타켓 : 해외 여행객 + 한국인 + 글로벌
• 목적 : 해외 친구 사귀기 (ex 여행 가는 곳의 현지인 친구)
• 핵심 기능
  ‣ 애플리케이션(사용자)
    1. 채팅 서비스
	- 사용자 – 사용자 채팅 서비스
	- 번역 서비스
	- 문법 교정 서비스
    2. 위치기반 서비스
	- 자신의 위치에 자신의 아바타 표시
	- 친구인 사용자와 상호작용 가능 (넣으면 좋을듯)
    3. 사용자 인중
	- 한국은 전화번호 인증 기반 -> 외국인 인증 불가 -> 여권, 신분증 등으로 인증
 	- 여권, 신분증 인증 후 사용자와 확인하기 이해 얼굴 인식 AI를 통해 2차 인증
 	- 인증 꼼꼼히 -> 안전하게 친구 사귀기 가능
    4. 소셜미디어 커뮤니티 기능 (인스타, 스레드 등)
 	- 사진, 동영상, 글 올리는 서비스
	- 다른 사용자가 게시하는 사진, 동영상, 글 볼 수 있음
	- 게시물에 상호작용 가능(댓글, 좋아요, 저장 등)
    5. 온라인 상품(선물하기) 서비스 (B2B2C-Business to Business to Consumer : 기업이 다른 기업을 통해 소비자에게 제품 or 서비스 제공)
	- 최종 소비자 : 제품 구매하고 사용
	- 애플리케이션 내에서 선물 구매 후 바로 친구에게 보내기
	- 상품 구매 서비스 + 구매 후 배송까지 연결
	- 상품 전달 서비스
	- 결제 서비스

  ‣ 웹 사이트(관리자)
    1. 사용자 관리
	- 사용자 회원가입 승인
	- 신고된 사용자 후처리 (넣으면 좋을 듯)
    2. 광고 게시
	- 애플리케이션에 광고 게시 및 관리
    3. 구매 물품 관리 및 구매 처리
	- B2B 제공자 : 제품 제공하는 기업
	- B2C 중개자 : 기업의 물품 등록하여 소비자에게 제품 전달
